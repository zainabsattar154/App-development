# SpendWise

SpendWise is an Expo React Native expense tracker with Firebase Auth, Firestore-ready transaction services, bottom tabs, analytics, light/dark theme support, and a demo login.

## Open in VS Code

Open the folder

## Run

```bash
npm install
npm start
```

Then scan the Expo QR code with Expo Go, or press `a` for Android emulator.

## Demo Login

```text
Email: demo@spendwise.app
Password: SpendWise123
```

The demo account works without Firebase keys and uses in-memory sample transactions. For production Firebase login, replace the placeholders in `src/services/firebase.js`, enable Email/Password Authentication, and configure Firestore.
